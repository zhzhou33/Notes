+ ```c++
  #ifdef DLL_EXPORTS
  #define DLL_API __declspec(dllexport)
  #else
  #define DLL_API __declspec(dllimport)
  #endif
  ```

  + 对于普通的一个类，定义一个外部接口构造函数，构造这个类。
  + 如果需要调用这个类的函数接口，需要
    + 声明成 virtual
      + 虚函数表，能够定位到
    + 声明成内联函数 inline
      + 是直接在类里加载的
      + 如果是普通的函数，还需要去重定位实现的代码，不占用类的 size，存放在代码区
    + 继承同理
    + 使用只有纯虚函数的抽象类之所以不需要导出，是因为纯虚函数的虚表作用
  + 对于一个构造了类的接口函数，就不需要导出这个类了，通过接口函数返回的指针就可以进行操作
    + 即用户不能直接构造这个类，因为这个类不是外部的
    + 如果需要，需要给类声明成外部的
